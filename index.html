FAQ

How do I deploy this on Vercel?
  1. Unzip and keep index.html in the project root.
  2. On Vercel, click + New Project → Import and select the folder, or drag & drop it onto the dashboard.
  3. No build step required. Framework preset: Other. Output directory: ./
How do I change colors or branding?
Edit the CSS variables under :root at the top of the file. Update the .brand text and the hero copy.
Is this accessible?
Keyboard focus rings, readable contrast, and semantic HTML are included. Please add alt text for real screenshots.
